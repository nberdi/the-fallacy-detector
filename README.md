# The Fallacy Detector

An interactive static website for practicing logical fallacies and argument evaluation.

## Fallacies Covered

- Ad hominem
- Straw man
- False dilemma
- Slippery slope
- Circular reasoning
- Hasty generalization
- Appeal to emotion
- Appeal to authority
- Red herring
- Bandwagon
- Weak analogy
- Post hoc fallacy

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
